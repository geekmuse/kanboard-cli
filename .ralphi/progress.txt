## Codebase Patterns
- **PortfoliosResource pattern**: Plugin API resources follow the same pattern as core resources (TYPE_CHECKING guard, _client.call(), typed returns). dict-returning methods (get_portfolio_projects, get_portfolio_tasks, get_portfolio_task_count, get_portfolio_overview) do NOT use PluginPortfolio.from_api() тАФ they return raw dicts or {}. Use `result or {}` for dict fallback (not `if not result: return []`).
- **get_portfolio_by_name raises on False**: Unlike typical list methods that return [] on False, name-lookup methods raise KanboardNotFoundError on both None AND False (per AC).
- **Dual-backend CLI pattern**: Add `_get_backend(app_ctx)` that calls `_get_store()` for local (preserving test mocking) and `create_backend("remote", client=...)` for remote. Add `_is_remote_backend(app_ctx)` for early-return checks in commands that diverge. Remote dependency commands skip `_get_backend` entirely and call `client.portfolios.xxx` directly after resolving portfolio_id. For milestone CLI (US-009), apply same pattern.

- **Packaging fixed (fix commit)**: `pyproject.toml` previously listed `kanboard-sdk>=1.1.1` as a runtime dep and only packaged `src/kanboard_cli` in the wheel. This caused `pip install -e '.[dev]'` to reinstall `kanboard-sdk` from PyPI, shadowing `src/kanboard/`. Fixed by: (1) adding `src/kanboard` to `[tool.hatch.build.targets.wheel] packages`, and (2) removing `kanboard-sdk` from dependencies (this repo IS the SDK). `tests/conftest.py` also adds `src/` to the front of `sys.path` as a belt-and-suspenders guard.
- **Model pattern**: All plugin API models go in `src/kanboard/models.py` in the `# Plugin models (Phase 1)` section. Use `_int()`, `_float()`, `_parse_date()` helpers. Export via `__init__.py` imports + `__all__` (sorted, RUF022 requires sorted `__all__`).
- **RUF022**: `__all__` must be sorted lexicographically. "Plugin..." < "Portfolio..." < "Project..." because 'l' < 'o'.
- **Ruff RUF002**: Use ASCII hyphen-minus `-` in docstrings for ranges (0.0-100.0), not en-dash `тАУ`.
